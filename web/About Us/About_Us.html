What is the Purpose of our application?

The Game Rule Checker is an app designed for board and card game enthusiasts who would benefit from a central repository of game rules. With this app, the user is able to select a supported game and search for a rule by keyword. In addition, users will be able to log-in, aks their own questions, and submit their own house rules.





Features of our Application

  1. Pages with various rules for board and card games
  2. Ability to add Rules and have it approved by our team
  3. Users can search for various rules through a functioning search bar
